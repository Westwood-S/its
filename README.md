# ITS SPIDER

## 1.安装

1. 安装**python3**(勾选Add python to environment variables)
1. 安装**requests**库: 在cmd中运行`pip install requests`

## 2.配置

配置**configure.json**:

    "time":抓取信息的间隔
    "email": 发送更新消息的邮箱
    "password": ITS的密码
    "username": ITS的帐户名

## 3.运行

命令行中运行: `python main.py`
